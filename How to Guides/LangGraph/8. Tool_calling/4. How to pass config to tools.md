
### ğŸ” **Code Explanation:**

âœ… `pets: List[str]` â†’ Accepts a list of pet names  
âœ… `config: RunnableConfig` â†’ Provides user ID at runtime  
âœ… `user_to_pets[user_id] = pets` â†’ Stores the user's pets in a dictionary
